/* 
 * README: Setup Instructions for Java Expense Tracker
 * 
 * 1. Database Setup:
 *    - Open MySQL Workbench or Command Line.
 *    - Execute the code in 'database.sql' to create the database and table.
 * 
 * 2. Project Setup:
 *    - Open Eclipse OR IntelliJ IDEA.
 *    - Create a New 'Dynamic Web Project'.
 *    - Copy the 'src' contents into your project's Java Resources/src.
 *    - Copy the 'WebContent' contents (JSP, CSS) into your project's WebContent or webapp folder.
 * 
 * 3. Dependencies:
 *    - Add 'mysql-connector-java.jar' to your project's WEB-INF/lib folder.
 *    - Add 'servlet-api.jar' (usually provided by Tomcat) to your build path.
 * 
 * 4. Configuration:
 *    - Open 'DBConnection.java' and update your MySQL username and password.
 * 
 * 5. Run:
 *    - Right-click on the project -> Run As -> Run on Server (Apache Tomcat).
 */
